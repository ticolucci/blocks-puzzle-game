## 4. Bonus points when the grid gets completely empty
When the user puts a piece that clears all existign blocks, the user gets bonus points.

## 5. Clearing a row or a column
After the user places a piece, if all slots of a column, or row of the grid are filled with blocks, then that row or column explodes out of the game (meaning that the column or row gets cleared of blocks) and the uses scores points.
All rows and all columns are cleared at the same time, so that both rows and columns might get cleared together in case of a piece completing both at the same time
The score is 1000 times the number of rows and columns cleared.
Times a multiplier according to the rules:
2 rows or columns together: x1.5
3 rows or columns together: x2
4 rows or columns together: x4
row and a column together: x2
